Hackathon Problem Statements

1. Juris AI

Skills Gained:

-Retrieval Augmented Generation technology
- Handling domain-specific terminology 
- Building a user-friendly interface for legal professionals 
- Interpreting and applying legal rules from retrieved documents 

2. Uwu-Forge

Skills Gained:

- Code Generation with LLMs: Training an LLM to produce syntactically correct and (ideally) semantically meaningful code. 
- Esoteric Language Modeling: Working with the unique syntax and constraints of the UwU language. 
- Creative Prompt Engineering: Formulating prompts that effectively guide the LLM to generate the desired UwU code. 
- Understanding of UwU Language Constructs: Implementing core UwU features like loops, conditionals, and error handling.

3. Note Ninja

Skills Gained:

- Audio processing and transcription through LLMs
- Summarization with LLMs 

4. Sensei Search

Skills Gained:

Leverage pre-trained open-source multimodal AI models
- Contextual understanding of queries combining modalities  
- Integrate with the RAG framework to provide accurate and context-rich results.

6. RAG-Narok

- Building a chatbot or interactive assistant capable of accurate, multimodal, and context-sensitive responses.
